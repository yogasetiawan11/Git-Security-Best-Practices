## RBAC
RBAC allows you to grant different levels of access based on roles within your organization. For example, DevOps engineers can have admin access, developers can have write access, and QA engineers can have read access 

## How to Implement RBAC:
Click Repository you work in, You can invite collaborators to your repository and assign them specific permissions (read, write, or admin) during the invitation process. This ensures that individuals have only the necessary access, contributing to better security practices.
